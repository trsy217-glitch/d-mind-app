# D_mind · 나의 성장 파트너 (PWA)

물방울 마스코트 **또로**와 함께 매일을 성장으로 만드는 일상 파트너 앱입니다.
설치 가능(Installable) + 오프라인 동작 + 상태 저장(연속일·포인트·물 트래킹 유지)이 되는 **실제 동작 PWA**예요.

## 폴더 구조
```
dmind/
├─ index.html            ← 앱 본체 (모든 기능 포함)
├─ manifest.webmanifest  ← 설치 정보
├─ sw.js                 ← 서비스워커 (오프라인 캐싱)
├─ api/
│  └─ claude.js          ← AI 프록시 (Vercel 서버리스 함수, 키 서버 보관)
└─ icons/                ← 앱 아이콘
```

## 동작하는 기능
- **또로 AI 코치 (PRO 핵심)** — 내 데이터(물·루틴·기분·독서·연속일)를 아는 대화형 코치. 홈에서 또로를 탭하거나 PRO 화면에서 진입. 무료 3회 체험, PRO 무제한. 미리보기에선 실제 Claude, 배포 시 규칙 기반 폴백
- **물 트래킹 / 독서 트래커** — 한 잔·15분 단위 기록, 목표 보너스, 날짜별 저장
- **루틴 체크** — 기상/아침/오후/운동/잠자리 완료 → 포인트·달성률 링 반영
- **성장 다이어리** — 기분 + 한 줄 + 그날 데이터 스냅샷을 기록으로 축적
- **이번 주 성장 리포트** — 7일 추세 차트 + 또로의 AI 주간 피드백
- **포인트·레벨·연속일** — XP 레벨업 연출, 매일 접속 시 연속일 자동 계산
- **식사 분석** — 사진 촬영 → 칼로리·혈당 영향·영양소·코치 조언
- **몸 인식 운동 (실시간 자세 추적)** — TensorFlow.js MoveNet으로 관절을 인식해 실제 스쿼트 횟수·자세 정확도 계산 + 라이브 스켈레톤. 라이브러리가 막힌 환경에선 데모 모드로 폴백
- 데이터는 기기 `localStorage`에 저장 (서버 불필요·프라이빗)

## 바로 띄워보기 (로컬)
PWA는 `file://`가 아닌 `http(s)://`에서 동작합니다.
```bash
cd dmind
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080
```

## Vercel 배포
```bash
cd dmind
vercel        # 또는 깃 연동 후 자동 배포
```
별도 설정 없이 정적 호스팅으로 바로 동작합니다. 배포 후 모바일 브라우저에서
"홈 화면에 추가"를 누르면 앱처럼 설치됩니다.

## AI를 진짜로 켜기 (서버리스 프록시)
앱의 AI 기능(식사 분석·또로 코치·주간 피드백)은 브라우저에서 `/api/claude`를 호출하고,
그 함수가 서버에서 비밀 키를 붙여 Anthropic으로 중계합니다. **키는 클라이언트에 노출되지 않아요.**
`/api/claude`가 없는 로컬/미리보기에서는 자동으로 규칙 기반 폴백으로 동작합니다.

켜는 순서:
1. Vercel에 배포 (`vercel` 또는 깃 연동). `api/claude.js`는 자동으로 서버리스 함수가 됩니다.
2. Vercel 대시보드 → Project → Settings → Environment Variables 에 등록:
   - `ANTHROPIC_API_KEY` = `sk-ant-...` (필수)
   - `CLAUDE_MODEL` = `claude-haiku-4-5-20251001` (선택, 기본값. 비용↓·비전 지원)
3. 재배포(Redeploy). 이제 실제 디바이스에서 AI가 진짜로 동작합니다.

비용 메모: 모델은 함수에서 **서버측으로 고정**해 비용을 통제합니다(클라이언트가 모델을 못 바꿈).
식사·코치 모두 Haiku 4.5로 충분하며 사진 1장 분석 ≈ 수 원 수준입니다.
운영 시 호출 제한(레이트리밋)은 별도 저장소(KV/DB)로 붙이는 것을 권장합니다.
